Media Query

It is used to create a responsive template.

Responsive UI

  1. Resolution (Width * height)
  2. Device type (desk, tab, mob)
  3. Orientation(portrait, landscape)
  4. Screen size(small=> mobile, medium => tablet, large => Desktop)
  5. Color Depth
  6. Pixel Depth
